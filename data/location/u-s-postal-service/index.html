Post Offices and Zipcode Boundaries

Download Data Package

United States Postal Service Data


Data Type: GIS Data Layer
Steward(s): AGRC/USPS

Abstract:
These datasets are related to mail delivery. Included are post office locations and zipcode boundaries in Utah.

Data Content:
The following data is available for download:

USPS Post Offices

[tabs]
[tab title="Description"]Location.ZipcodePOBoxes represents the approximate locations of USPS Post Offices in Utah. Each zipcode's assigned post office has a point, therefore if a post office has multiple zipcodes, it will have multiple points.

[/tab]

[tab title="Usage"]
The native spatial reference for this dataset is UTM Zone 12N, NAD83 (0.01 meter coordinate precision).

There are no constraints or warranties with regard to the use of this dataset. Users are encouraged to attribute content to: State of Utah, SGID.[/tab]

Zip Code Boundaries

[tabs]
[tab title="Description"]Boundaries.ZipCodes represents an ongoing effort to approximate the geographic extents of five digit zip codes. The dataset was produced using a combination of methods and is based on several sets of source data. Editing methods include:

1) Using local zipcode polygon data obtained from counties and cities.
2) Identifying place locations (cities, towns, populated areas, etc.) from the postal service website and address information system (AIS) and building zipcode boundaries around these places.
3) Using the 2000 Census TIGER line fileâ€™s zip code attributes.

In addition, AGRC has used the locations of mailing addresses known to be valid to fine tune this dataset.

[/tab]

[tab title="Usage"]
The native spatial reference for this dataset is UTM Zone 12N, NAD83 (0.01 meter coordinate precision).

There are no constraints or warranties with regard to the use of this dataset. Users are encouraged to attribute content to: State of Utah, SGID. [/tab]
