Download Topographic Map Indices

Topographic Map Indices


Data Type: GIS Data Layer
Steward: AGRC

Abstract:
This dataset contains GIS mapping data representing the tile indices for the 24K, 100K, and 250K USGS scanned topographic maps (DRGs).

Data Content:
The following data is available for download:

USGS 24K DRG Indices


The USGS Topographic Map tile indices can be used to determine the 7.5 minute, 1:24,000 scale map boundaries for Utah.

Each tile index contains the following attribute fields:

  • Name - the name of the map
  • OHIO_CODE - code usded by USGS
  • TILE - the image tile name on FTP site
  • PATH - the directory where the topographic map tile resides on the FTP site

The 24K USGS topographic map tile index is maintained by AGRC. Changes and updates are not planned to this dataset.

The native spatial reference for this dataset is UTM Zone 12N, NAD83 (0.01 meter coordinate precision).

There are no constraints or warranties with regard to the use of this dataset. Users are encouraged to attribute content to: State of Utah, SGID.
This dataset is maintained by AGRC.

Questions can be direction to Rick Kelson (801.538.3237) at AGRC.

    USGS 100K DRG Indices
