DEM and NED Indices


Data Type: GIS Data Layer
Steward: AGRC

Abstract:
This dataset contains GIS mapping data representing the tile indices for the USGS DEM and NED elevation datasets for Utah.
Data Content:
The following data is available for download:

USGS Digital Elevation Model (DEM) Indices

The USGS 24K tile index can be used to determine the 7.5 minute, 1:24,000 scale boundaries of the 10, 30, and 90 meter USGS DEMs for Utah.

Each tile index contains the following attribute fields. The 10 and 30 meter USGS DEMs are named according the OHIO_CODE attribute:

  • Name - the name of the map
  • OHIO_CODE - code used by USGS
  • TILE - the image tile name on FTP site
  • PATH - the directory where the topographic map tile resides on the FTP site

The 24K tile index is maintained by AGRC. Changes and updates are not planned to this dataset.

The native spatial reference for this dataset is UTM Zone 12N, NAD83 (0.01 meter coordinate precision).

This dataset is maintained by AGRC.

USGS National Elevation Dataset (NED) Indices

The USGS 24K tile index can be used to determine the 7.5 minute, 1:24,000 scale boundaries of the 10 and 30 meter USGS NED tiles for Utah.

Each tile index contains the following attribute fields. The 10 meter USGS NED tiles are named according the TILE attribute:

  • TILE - the image tile name on FTP site
  • PATH - the directory where the topographic map tile resides on the FTP site

In addition to 10 meter NED available by 24K quad, there are county mosaics of the 10 and 30 meter datasets.

The 24K, 10 meter quad tile index is maintained by AGRC. Changes and updates are not planned to this dataset.

The native spatial reference for this dataset is UTM Zone 12N, NAD83 (0.01 meter coordinate precision).

There are no constraints or warranties with regard to the use of this dataset. Users are encouraged to attribute content to: State of Utah, SGID.
