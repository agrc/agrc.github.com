AGRC's ArcSDE database server contains SGID's vector-format GIS feature classes.

ArcMap users can make a live connection to the ArcSDE Server using the public, read only user and password shown below.

Performance is highly dependent on the speed of the user's downstream internet connection. Greater than 10 Mbps is recommended.

When to use?
Using the a database connection to the most current ArcSDE server can be advantageous as users will see and query the most updated version of all feature classes housed at AGRC. Live connections are excellent for browsing, exploring, or making quick maps with SGID data layers. However,  users are encouraged to download or export data files where editing, archiving project data, or optimal performance is required. Additionally, it is recommended and in some cases required that users' ArcMap release version be at or above the level of AGRC's ArcSDE server.

How to directly connect to the SGID ArcSDE 10.0 Server: 

1. In ArcCatalog browse in the Catalog Tree to the option "Add Spatial Database Connection"


2. Provide the connection information you see below. The password is 'agrc'.

3. Click on the connection you just established to select, preview and add ArcSDE-based feature classes of your choice.
SGID Organization

The SGID's organization is based on ISO Topic Categories. AGRC treats the ISO categories as a guideline, and the main goal is to maximize usability for GIS users.
