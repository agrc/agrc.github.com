Local Parks

Data Type: GIS Data layer
Steward: AGRC

Abstract:
This dataset shows the locations of local parks excluding State and Federally owned parks.
This dataset also contains the name, county, city, acres, type and construction status of the parks.

Usage: There are no constraints or warranties with regard to the use of this dataset.
