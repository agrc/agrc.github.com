Golf Courses

Data Type: GIS Data Layer
Steward(s): AGRC

Abstract:
This dataset represents golf course locations within the State of Utah. It should be noted that this is based on the Utah Golf Association's website list of golf corses in Utah and may be incomplete.
This dataset also contains the name, city, number of holes, par, and type of gold course.

Usage: There are no constraints or warranties with regard to the use of this dataset.
