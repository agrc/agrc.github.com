Concept of an Address Locator

Download Address Locators

Web API Address Locators

  • Data Type: Address Locator Service
  • Steward(s): AGRC
  • Web Application: https://api.mapserv.utah.gov
  • Abstract: AGRC offers a geocoding web service API to return geographic coordinates for Utah addresses using the SGID Transportation.Roads and Location.AddressPoints feature classes as reference data. This API can be accessed over http programmatically (with standard HTTP get request) without the use of GIS-specific software - such as a web application. This geocoding API uses 'front end' logic to resolve several common address standardization issues above and beyond what out-of-the-box geocoders handle. The downloadable ArcGIS Locator Packages described below are copies of the 'back end' services that are used directly by the geocoding API.
  • Using the API Locators:
    For geocoding within a programming environment, it is recommended that you use the geocoding API web service.
    Within ArcGIS, the geocoding API can be accessed using our custom geoprocessing tool.
    Additionally, we offer a Google Sheets Add-on (via Chrome browser) that provides geocoding directly within a spreadsheet. This can be used by anyone with a utah.gov email address.

ArcGIS Desktop Locators

  • Data Type: ArcGIS Locator Package (.gcpk)
  • Steward(s): AGRC
  • Abstract: AGRC also offers traditional ArcGIS Desktop address locators. These are downloadable address locators that are built using the SGID Transportation.Roads and Location.AddressPoints feature classes as reference data. These are the 'back end' locators for the Web API address locators described above. They are rebuilt monthly based on the statweide roads update schedule.

Package Contents

The following data is available for download:

Roads Address Locator with Zip Code Zone

Locator_RoadsZip_COMPOSITE.loc is an ArcGIS 10 composite address locator that references the statewide Transportation.Roads feature class and uses the ZIP_LEFT and ZIP_RGHT fields for zone. This address locator has four levels of street name aliasing to include alternate street names and address grid coordinate system names. For example, Redwood Road is aliased as both Highway 68 and 1700 West.

Roads Address Locator with Address Quadrant System Zone

Locator_RoadsAddrSys_COMPOSITE.loc is an ArcGIS 10 composite address locator that references the statewide Transportation.Roads feature class and uses the address quadrant system for zone. This address locator has four levels of street name aliasing to include alternate street names and address grid coordinate system names. For example, Redwood Road is aliased as both Highway 68 and 1700 West. By using the address quadrant system for zone, applications and users will be able to match addresses even if the city or zip code supplied is close but not exact or when a location is at or near the border of two zones.

Address Points Locator with Zip Code Zone

Locator_AddressPtsZip.loc is an ArcGIS 10 address locator that references the Location.AddressPoints feature class and uses zip code for zone. Geocoding match rates will improve with the Locator_AddressPtsZip.loc address locator as the Location.AddressPoints feature class continues to be developed.
