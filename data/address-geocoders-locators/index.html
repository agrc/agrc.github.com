Concept of an Address Locator

Download Address Locators

Web API Address Locators

  • Data Type: Address Locator Service
  • Steward(s): AGRC
  • Web Application: https://api.mapserv.utah.gov
  • Abstract: AGRC offers a geocoding web service API to return geographic coordinates for Utah addresses using the SGID Transportation.Roads and Location.AddressPoints feature classes as reference data. This API can be accessed over http programmatically (with standard HTTP get request) without the use of GIS-specific software - such as a web application. This geocoding API uses 'front end' logic to resolve several common address standardization issues above and beyond what out-of-the-box geocoders handle. The downloadable ArcGIS Locator Packages described below are copies of the 'back end' services that are used directly by the geocoding API.
  • Using the API Locators:
    For geocoding within a programming environment, it is recommended that you use the geocoding API web service.
    Within ArcGIS, the geocoding API can be accessed using our custom geoprocessing tool.
    Additionally, we offer a Google Sheets Add-on (via Chrome browser) that provides geocoding directly within a spreadsheet. This can be used by anyone with a utah.gov email address.

ArcGIS Desktop Locators

  • Data Type: ArcGIS Locator Package (.gcpk)
  • Steward(s): AGRC
  • Abstract: AGRC also offers traditional ArcGIS Desktop address locators built using the SGID Transportation.Roads and Location.AddressPoints feature classes as reference data. These are the 'back end' locators for the Web API address locators described above - made available for download. We offer a roads locator and an address point locator, plus a composite. They are rebuilt monthly based on the current statewide roads update schedule.
  • Using the Desktop Locators:
    AGRC's locators are available for download via 'Locator Packages' (Locator Packages are not supported in ArcGIS 10.0 or earlier). These are portable files that do not require you to download the reference data: the data is packaged. Once downloaded, you simply 'Unpack' and it is ready for use. You can 'Unpack' the Locator Package in ArcCatalog by right-clicking on the package. Unpacking will always place the unpacked locator in your user profile ('..\My Documents\ArcGIS\Packages'). Alternatly, you can use ESRI's 'Extract Package' geoprocessing tool and specify an output location. You can also drag the .gcpk file into ArcMap and it will unpack.

Package Contents

The following data is available for download:

Roads Address Locator

AGRC_RoadsLocator is an ArcGIS 10 address locator that references the statewide Transportation.Roads feature class and uses the right and left zip code fields and/or the right and left address systems fields for zone. This address locator has four levels of street name aliasing to include alternate street names and address grid coordinate system names. For example, Redwood Road is aliased as both Highway 68 and 1700 West.

Address Points Address Locator

AGRC_AddressPointLocator is an ArcGIS 10 address locator that references the statewide Location.AddressPoints feature class and uses zip code or address system for zone. This address locator has four levels of street name aliasing to include alternate street names and address grid coordinate system names. For example, Redwood Road is aliased as both Highway 68 and 1700 West. By using the address quadrant system for zone, applications and users will be able to match addresses even if the city or zip code supplied is close but not exact or when a location is at or near the border of two zones.

Questions can be directed to Greg Bunce (801.538.3665) at AGRC.

Composite Address Locator

AGRC_CompositeLocator is an ArcGIS 10 composite address locator that references the Transportation.Roads and Location.AddressPoints feature classes and uses zip code or address system for zone.
