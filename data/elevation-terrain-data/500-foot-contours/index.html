500 Foot Contours


Description
ELEVATION.Contours500Ft is a statewide dataset of contour lines at a 500 foot interval for cartography and approximate elevation identification.

Usage
Elevation information is stored in the 500 Foot contour lines in an attribute ELEV associated with each contour isoline.

The native spatial reference for this dataset is UTM Zone 12N, NAD83.

There are no constraints or warranties with regard to the use of this dataset. Users are encouraged to attribute content to: State of Utah, SGID.
