Grazing Improvement and Allotment Areas


Data Type: GIS Data layer
Steward(s): USDA & AGRC

Abstract:
These datasets contain statewide Grazing Improvement Program and grazing allotment boundaries for use in planning and land management.

Data Content:
The following data is available for download:

Grazing Improvement Areas


Description
Farming.GrazingImprovementRegions contains statewide Grazing Improvement Program boundaries for use in planning and land management.

Usage
