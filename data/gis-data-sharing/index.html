GIS Data Sharing


Why? bring the world into harmony
How? should we make a drop box, email, pull from SDE/FTP/Dropbox
What? listed below are the layers and desired attributes

Road Centerlines

  • Street name
  • Street type
  • Prefix & Suffix directions
  • Address ranges

Address Points

  • Full Address (ex. 622 N KAYCIE LN)
  • City
  • Zip Code
  • Parcel ID

Parcels

  • Parcel ID
  • Parcel Address (optional)
