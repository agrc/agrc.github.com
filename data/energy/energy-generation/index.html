Power Plants (CO2)

Data Type: GIS Data Layer
Steward: AGRC

Abstract:
This dataset contains Power Plant locations from CO2 project for the State of Utah, July 2008.
This dataset also contains the ORIS Code, name, type, CO2 Annual Tons and fuel code for the plants.
