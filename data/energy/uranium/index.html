Uranium Data

Download Package

Uranium Data


Data Type: GIS Data Layer
Steward: AGRC

Abstract:
This dateset contains the locations permitted uranium mines, current uranium mills, uranium past producers and uranium area boundaries and districts for the State of Utah.

Data Content:
The following data is available for download:

Permitted Uranium Mines


Description
This dataset contains the locations of permitted uranium mines in Utah in addition to a mine ID, type, status, name, operator, product and more.
