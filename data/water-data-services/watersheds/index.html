Watershed Areas

Data Type: GIS Data Layer
Steward(s): AGRC, BLM

Abstract:
Water.Watersheds_Area represents the watershed areas throughout Utah. This data set is intended to be used as a tool for water-resource management and planning activities, particularly for site-specific and localized studies requiring a level of detail provided by large-scale map information.

Usage:
The HUC_8 field contains the 8 digit hydrologic unit code for the 4th level sub-basins
The HU_8_NAME field contains the sub-basin name
The HUC_10 field contains the 10 digit hydrologic unit code for the 5th level watersheds
The HU_10_NAME field contains the watershed name
The HUG_12 field contains the 12 digit hydrologic unit code for the 6th level subwatersheds
The HU_12_NAME field contains the subwatershed name
