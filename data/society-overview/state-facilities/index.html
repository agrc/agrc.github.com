Locations of State Facilities

Download Dataset

State Facilities

Data Type: GIS Data Layer
Steward(s): AGRC & Other State Agencies
Web Application: State Facilities Application

Abstract:
Society.StateFacilities contains locations for state owned or leased buildings. This dataset contains a wide and varied collection, some schools in the k-12 system and also higher education buildings, Office buildings, port of entries, rest areas and fish hatcheries. It also contains many other facilities such as UDOT sheds, restrooms, and other smaller facilities. The web application showing this data, also shows other facility data to give context. For users wanting a full Schools data layer please see Society.Schools.
