Updates were made recently to the SGID10.Society.Schools feature class that resides on the Utah SGID ArcSDE database server.

The updated Schools data is also available as shapefiles and file geodatabase files (Libraries are included in this data package) for download on the SGID FTP site.

Based on information provided by the Utah State Office of Education (as of July 2012), the update includes enhancements to the geographic location of schools, address, district information, internal ids, grade levels, and new point locations of schools built since the dataset was originally created in November 2010.

Additionally, the Schools data is now being stored in a more intutive and user friendly schema. The new schema now includes fields that indicate which Utah House and Senate District each school is located in.
