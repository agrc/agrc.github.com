Neighborhood Suitability Analysis in Utah

Suitability analysis, using GIS, allows a user to spatially explore the appropriateness of an area for the specific criteria that you define. This application highlights suitable areas on the map based on the weights given in the interactive slider bars. Darker hexagons indicate areas of higher suitability. Conversely, this map can be used to explore areas that are less suitable.

This map was created using OSM and the Uber H3 Hexagonal Geospatial Indexing System, resolution 8.
