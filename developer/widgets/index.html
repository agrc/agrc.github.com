Save yourself some time and use our widgets!

AGRC is actively developing and maintaining a core set of geospatial javascript widgets and ArcMap Addons. These widgets allow for developers and non-developers to incorporate geospatial functionality into their website, blog, or GIS environment. All of our widgets and addons are powered by our geospatial web APIs that you can also use to create your own addons and wigdets.

A great place to get started with our javascript widgets is the GitHub repository. Checkout the test pages to see the widgets in action. Also, all of our public websites are using the widget library so you may already be familiar with them!

In a nutshell, our widget library allows you to have a user locate things by
[list style="list4"]

  • Geocode their address with the FindAddress Widget - Type in an address and get a UTM or latitude longitude coordinate, optionally zooming a map to the location.
  • Find anything based on an attribute with the FindGeneric widget - Zoom a map to any layer in our SGID. eg: Zoom to county or municipality.
  • Find valid township, range, and sections with the TRSsearch widget - Zoom a map or get the formatted TRS value from this widget.

[/list]

You can also quickly add common mapping elements: map, base map selector, etc to your website or blog.
