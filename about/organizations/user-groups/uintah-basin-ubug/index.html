Unitah Basin User Group: UBUG consists of Geo-spatial technicians and specialists from State and Federal agencies, private organizations, Uintah, Daggett, Duchesne, and Wasatch Counties. It is open to anyone that would like to attend. This user group conducts quarterly meetings featuring trainings, discussions on GIS and other Geo-spatial applications and/or problems that those attending might need help with.
